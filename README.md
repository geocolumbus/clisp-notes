# Common LISP Notes

## Install Common LISP on MacOS
```
brew install clisp
clisp --version
```

## Usage

### Install
```
git clone https://github.com/geocolumbus/clisp-notes.git
cd clisp-notes
```

### Run
```
clisp clisp-notes.lisp
```
